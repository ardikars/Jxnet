# Jxpcap
Pcap is a java project that comes with a required native shared library (JNI/Java Native Interface).

## Requirement

JDK 

PCAP:

Windows (Winpcap)

Linux (Libpcap)

## Install


### Windows

make -f Makefile.win

make -f Makefile.win install


### Linux
make

make install
